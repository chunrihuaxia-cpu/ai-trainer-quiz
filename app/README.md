# 📝 AI训练师 · 刷题 App

人工智能训练师（三级）理论知识复习题刷题工具，适用于 iOS / Android。

## 功能

- **900道完整题库**：判断题(300) + 单选题(300) + 多选题(300)
- **分类刷题**：按题型筛选，支持乱序模式
- **智能判题**：正确自动跳下一题，错误显示解析
- **错题复习**：自动记录错题，支持针对性复习
- **PWA 支持**：可添加到 iOS 主屏幕，像原生 App 一样使用
- **离线可用**：首次加载后可离线刷题

## 快速开始

### 方法一：本机测试（Mac）

```bash
cd app/
python3 -m http.server 8080
```

然后在浏览器打开 `http://localhost:8080`

### 方法二：iPhone 访问（同一 WiFi）

```bash
cd app/
# 获取本机 IP
ifconfig | grep "inet " | grep -v 127.0.0.1
# 假设 IP 是 192.168.1.5
python3 -m http.server 8080
```

在 iPhone Safari 打开 `http://192.168.1.5:8080`，点击分享 → "添加到主屏幕"

### 方法三：部署到 GitHub Pages（免费）

1. 将 `app/` 目录下的文件上传到 GitHub 仓库
2. Settings → Pages → 选择 main 分支
3. 在 iPhone 上访问 `https://你的用户名.github.io/仓库名/`

## 使用说明

| 题型 | 操作 |
|------|------|
| 判断题 | 点击「正确」或「错误」按钮 |
| 单选题 | 点击选项 A-E |
| 多选题 | 点选多个选项后，点击「确认提交」 |

- 答对 → 自动跳到下一题
- 答错 → 显示正确答案 + 解析，点击「下一题」继续
- 完成一轮后可「复习错题」或「重新开始」
- 点击「🔀 乱序」切换随机/顺序模式
